Dashboard for World Development Indicators

  • Fr 11 Januar 2019
  • Back
The most current and accurate global development data of our world is provided online by [The World Bank](https://datacatalog.worldbank.org/dataset/world-development-indicators). Unfortunately, one quickly becomes lost after downloading this dataset as there are over 10000 development indicators available for each country. To obtain a better overview of this dataset I programmed a dashboard that lets you explore the different indicators using a choropleth map. Additionally, the development of two countries over time can be compared directly in a scatter plot.

Resources¶

The dashboard for the world development indicators consists of a multi-page app that programmed using [dash](https://plot.ly/products/dash/). The code for this project is available [here](https://github.com/danielsdatahacks/wdi-app-db_github). To run this dashboard locally in your browser just install the packages in the requirements.txt and run the command "python index.py" in your console. Currently, a simplified version of the dashboard showing only indicators of type "emission" and "energy" is hosted on [heroku](https://wdi-app-db.herokuapp.com) (Username: WDI_app, Password: 001D73). Notice that this dashboard is not designed for small smartphone screens.

1. Explore¶

The first part of my dashboard is called "Explore". It lets you quickly explore the data in a choropleth map to compare all countries for which the data is available.
